# 微信登录功能实现总结

## 已完成的工作

### 1. API 层更新 (`src/api/wechat.js`)

**新增功能：**
- ✅ `loginWithCode()` - 使用微信 code 快速登录
- ✅ `loginWithPhone(phoneCode)` - 使用手机号授权登录
- ✅ `getPhoneNumber(e)` - 处理手机号授权事件

**登录流程：**
1. 调用 `wx.login()` 获取登录 code
2. 将 code 发送到云函数 `auth-user`
3. 云函数验证 code 并返回用户信息
4. 前端保存用户信息到 store 和本地存储

### 2. Store 层更新 (`src/store/user.js`)

**新增方法：**
- ✅ `wechatLogin()` - 微信登录
- ✅ `phoneLogin(phoneCode)` - 手机号登录
- ✅ `checkLoginStatus()` - 检查登录状态
- ✅ `ensureLogin()` - 确保用户已登录（用于需要登录的功能）

**登录状态判断：**
```javascript
isAuthenticated: (state) => !!state.openid && !!state.isLoggedIn
```

### 3. 登录弹窗组件 (`src/components/LoginModal.vue`)

**功能特性：**
- ✅ 支持两种登录方式：
  - 📱 微信手机号登录（需要用户授权）
  - ⚡ 快速登录（使用 code）
- ✅ 登录中状态显示
- ✅ 登录成功后自动关闭并触发回调
- ✅ 用户可取消登录

**使用方法：**
```vue
<LoginModal
  :show="showLoginModal"
  message="使用 AI 功能需要先登录"
  @close="showLoginModal = false"
  @success="handleLoginSuccess"
/>
```

### 4. 诊断页面更新 (`src/pages/diagnose/diagnose.vue`)

**登录检查流程：**
1. 用户点击"开始诊断"按钮
2. 调用 `userStore.ensureLogin()` 检查登录状态
3. 如果未登录，显示登录弹窗
4. 登录成功后自动继续诊断流程

**关键代码：**
```javascript
async function startDiagnose() {
  // 检查登录状态
  const isLoggedIn = await userStore.ensureLogin()
  if (!isLoggedIn) {
    showLoginModal.value = true
    return
  }

  // 继续诊断流程...
}

function handleLoginSuccess() {
  // 登录成功后自动开始诊断
  startDiagnose()
}
```

### 5. 识别页面更新 (`src/pages/add-plant/add-plant.vue`)

**登录检查流程：**
1. 用户点击"AI 识别"按钮
2. 调用 `userStore.ensureLogin()` 检查登录状态
3. 如果未登录，显示登录弹窗

## 登录流程图

```
用户点击 AI 功能按钮
    ↓
检查登录状态 (ensureLogin)
    ↓
已登录? ──Yes──→ 继续功能流程
    ↓ No
显示登录弹窗
    ↓
用户选择登录方式
    ├─ 手机号登录 → 授权 → 获取 phoneCode → 调用云函数
    └─ 快速登录 → 获取 code → 调用云函数
    ↓
云函数验证并返回用户信息
    ↓
保存用户信息到 store 和本地存储
    ↓
触发 @success 回调
    ↓
自动继续之前的功能流程
```

## 云函数要求

### auth-user 云函数需要支持的 action：

1. **wechatLogin** - 微信 code 登录
   ```javascript
   {
     action: 'wechatLogin',
     code: 'wx_login_code'
   }
   ```

2. **phoneLogin** - 手机号登录
   ```javascript
   {
     action: 'phoneLogin',
     loginCode: 'wx_login_code',
     phoneCode: 'phone_auth_code'
   }
   ```

### 返回格式：
```javascript
{
  code: 200,
  message: '登录成功',
  data: {
    user: {
      _id: 'user_id',
      _openid: 'openid',
      username: 'username',
      nickname: 'nickname',
      avatar: 'avatar_url',
      phoneNumber: 'phone_number'
    },
    token: 'auth_token'
  }
}
```

## 注意事项

1. **手机号授权按钮**：必须使用 `<button open-type="getPhoneNumber">` 才能触发手机号授权
2. **登录状态判断**：使用 `openid` 和 `isLoggedIn` 双重判断
3. **本地存储**：登录信息保存在 `uni.storage` 中，应用启动时自动恢复
4. **云函数层**：Layer 已上传，包含共享的 CloudBase SDK 初始化代码
5. **错误处理**：云函数返回 401 时自动弹出登录窗
