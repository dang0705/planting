# Subagent 复用规则（单线程约束）

更新时间：2026-04-22

## 规则

1. 同一角色（包括 `new-rules-keeper`、`modulize-keeper`、`dent-guard`、`review` 等）在同一会话内必须复用同一个 subagent 线程。
2. 不允许并行启动同角色的多个线程。
3. 当需要继续该角色任务时，优先复用已存在的对应线程；若该线程失效，再重新启动同角色的新线程并关闭旧线程的残留。
4. 如确有特殊场景必须重开，必须在闭环前写明原因与替换依据。
