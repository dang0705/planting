# 香菜 - 常见问题汇总 - 阳台种菜

## 常见问题及原因
1. 抽薹：温度持续高于25℃
2. 黄叶：浇水不当、缺肥
3. 不发芽：播种温度过低、种子老化
4. 虫害（蚜虫）：高温干燥
5. 徒长：光照不足、氮肥过多

## 立即处理步骤（按问题分类）
- 抽薹：立即摘除花薹、遮阳降温
- 黄叶：控水、施稀释氮肥
- 不发芽：提高温度至18–25℃、更换新鲜种子
- 虫害：喷生物杀虫剂、增加通风
- 徒长：移至强光、暂停氮肥

## 预防措施
- 春秋播种，避免夏季
- 保持土壤湿润、见干浇水
- 每月检查虫害
- 每天至少6小时光照
