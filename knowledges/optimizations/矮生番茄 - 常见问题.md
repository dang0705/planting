# 矮生番茄 - 常见问题汇总 - 阳台种菜

## 常见问题及原因
1. 掉花：温度波动、缺硼钙
2. 裂果：浇水不均、果实膨大期雨水多
3. 叶黄：缺肥（缺氮/铁）、根部问题
4. 虫害（白粉虱/蚜虫）：高温干燥
5. 徒长：光照不足、氮肥过多

## 立即处理步骤（按问题分类）
- 掉花：人工授粉、喷钙硼叶面肥、稳温
- 裂果：均匀浇水、遮雨、套袋保护
- 叶黄：施均衡肥、检查根系、补光
- 虫害：喷杀虫剂、增加通风
- 徒长：移至强光、控氮肥

## 预防措施
- 保持20–28℃稳定温度
- 均匀浇水，避免忽干忽湿
- 每月检查虫害
- 每天6小时以上光照
